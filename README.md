# Components
WM: [Hyprland](https://hypr.land/)

Bar: [Waybar](https://github.com/alexays/waybar)

Shell: [Bash](https://www.gnu.org/software/bash/)

Color Generation Tool: [Matugen](https://github.com/InioX/matugen)

# Installation
Clone the repository and run `install.sh`.
```
git clone https://github.com/esperime/cozyforest.git
cd cozyforest
./install.sh
```

# Usage
`SUPER + ENTER` - Launch Terminal emulator

`SUPER + D` - Launch Fuzzel

`SUPER + E` - Open file manager

`SUPER + Q` - Quit / Exit

`SUPER + L` - Lock screen

`PRINT` - Capture a window

`SHIFT + PRINT` - Capture fullscreen

`SUPER + PRINT` - Capture a region
